# Copilot

# Usage

write custom instructions in a copilot-instructions.md file, and scale your workflow by saving reusable prompts in your .github/prompts folder.

- `.github/prompts/` - store reusable prompt templates here
  - e.g. `.github/prompts/write-tests.md` - "Write unit tests for this file, covering edge cases and error handling."
- `.github/copilot-instructions.md` - write custom instructions for your Copilot agent here
