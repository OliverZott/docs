# Ubuntu Filesystem

```bash
cd ~  # home directory
cd /  # root directory
tree -L3 /  # list directories and files in tree format (up to 3rd level beginning at root dir)
tree -L2 ~  # list directories and files in tree format (up to 2nd level beginning at home dir)
```
